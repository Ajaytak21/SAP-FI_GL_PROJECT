# SAP-FI_GL_PROJECT
End-to-end SAP FI General Ledger project in a simulated company code. Posted consolidated journal entries for long-term loans, mortgage repayment, investments, asset purchases, vendor settlements, and asset disposal. Included goodwill revaluation adjustments and validated results using G/L Balance and Financial Statement reports.

Objective

This project demonstrates execution of consolidated journal entries in SAP FI using transaction FB50 within a simulated company code environment.

* Business Transactions Covered
* Long-term loan receipt
* Mortgage repayment
* Long-term investment
* Equity share issuance
* Motor vehicle purchase
* Factory equipment purchase
* Payables settlement
* Furniture disposal
* Goodwill & patent revaluation
* SAP Tools Used
* SAP GUI
* FB50 – G/L Account Posting
* Financial Statement Report
* G/L Account Balance Report

Outcome

Successfully posted all transactions in a single accounting document and validated accuracy using SAP financial reports.
